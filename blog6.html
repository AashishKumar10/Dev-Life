Cara – My First Clothing
Store Website

After completing my family website project, I wanted to try something commercial — so I created Cara, a clothing store website. This was my first time designing a brand-style website with a modern product layout, attractive fonts, and user-friendly sections.

I focused on building sections like a homepage banner, featured products, new arrivals, and a clean footer. I used HTML and CSS to organize the content into grids and responsive blocks. I also added logo design for “Cara” to make the store feel real and unique.

One thing I learned in this project was visual balance — how to use whitespace, fonts like ‘Outfit’, and buttons to make the design simple but attractive. I practiced using Font Awesome icons, button hover effects, and product cards.

Though there’s no cart or backend, this project taught me how e-commerce homepages are structured, and it helped me gain confidence for future dynamic websites.

In this blog, I explain how I built each section step-by-step and how you can create your own brand-style homepage even as a beginner.
